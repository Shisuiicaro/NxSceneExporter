# NxScene Exporter (UPM)

Unity Editor exporter that generates a single `.nxscene` file (zip) containing platform AssetBundles + a `manifest.json`.

## Install (Git URL)

Add via Unity Package Manager:

- Window -> Package Manager
- + -> Add package from git URL...
- `https://github.com/<OWNER>/<REPO>.git?path=/Packages/com.nexus.nxscene-exporter#v1.0.0`

## Usage

Menu:

- Tools/Nexus/Export/Export Current Scene (.nxscene)
- Tools/Nexus/Export/Export Prefab Folder (.nxscene)
